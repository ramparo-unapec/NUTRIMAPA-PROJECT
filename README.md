# NUTRIMAPA

## Descripción del Proyecto

**NutriMapa** es una plataforma web de impacto social diseñada para reducir el desperdicio de alimentos y fortalecer la seguridad alimentaria en la República Dominicana.

La plataforma conecta a **donantes de alimentos** como restaurantes, supermercados y hoteles con **comedores comunitarios, voluntarios repartidores y familias en situación vulnerable** del Gran Santo Domingo, facilitando la gestión, distribución y aprovechamiento de excedentes alimenticios.

A través de una solución tecnológica, NutriMapa busca crear una red colaborativa que permita registrar donaciones, coordinar entregas y mejorar la trazabilidad del proceso de distribución de alimentos, promoviendo la solidaridad, sostenibilidad y apoyo comunitario.

## Objetivo General

Desarrollar una plataforma tecnológica que facilite la conexión entre entidades generadoras de excedentes alimenticios y comunidades necesitadas, optimizando la gestión de donaciones y contribuyendo a la reducción del desperdicio de alimentos.
